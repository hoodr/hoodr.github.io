Divvy

Check Splitter Application for Android Devices using Optical character recognition and the splitwise API to send instant payment requests. This is the final Project for CS3344 Mobile Application Development at Boston College

* More updates to come (will post wireframes here) *

For issues and inquires (looking at you splitwise), please email me at hoodr@bc.edu and I'll get back to you as soon as possible!

Project Proposal

This app will work in two steps. OCR for the check itself, and then the Splitwise api to request the money from each member of the party.

Behavior flow:

First the user to select the total number of people in the party, and then the contacts of those people. The app will then take a user to the camera, and the user will take a photo of the check. OCR will then extract the items and their prices, then ask the user to enter the gratuity and calculate the split tax and tip. Next, the user will take the itemized menu items, select which user it belongs to, then tap finish. The app will then automatically send a formatted Venmo request, detailing the items they ordered, and the total they owe, to each of the members of the party.
